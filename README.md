Overview
==========
URIs, indexes and RSS feeds of the largest 72 closed-source publishers... and the code to scrape them. 

Proof
=========
https://docs.google.com/spreadsheet/ccc?key=0AtOqyz8P_fJ0dHNKUmh4UGxsa1hVdXBKVmd3Zy0yc3c

Usage
==========
- $ cd OpSci/
- $ bundle install
- $ cd lib/scrapers
- $ ruby elsevier.rb
